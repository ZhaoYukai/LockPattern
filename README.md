# LockPattern
Android九宫格图案解锁<br>
<br>
刚进入程序的时候，会让用户设置一个密码，设置完毕后，程序会自动退出。<br>
再次进入程序的时候，就需要用户来画线解锁之前的图案了，当解锁成功的时候会自动跳到设置图案的界面。<br>
